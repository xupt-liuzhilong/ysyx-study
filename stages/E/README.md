## 记录方式

每个小阶段单独记录：

```text
logs/          每日学习日志
notes/         知识笔记
problems/      问题记录
screenshots/   运行截图
code/          练习代码
```

其中：

* `logs/` 记录每天做了什么
* `notes/` 记录知识点总结
* `problems/` 记录遇到的问题和解决过程
* `code/` 保存对应阶段的练习代码
* `screenshots/` 保存运行结果或关键过程截图
