## 学习重点

当前重点：

* 字符串基础与字符数组
* 指针基础与指针传参
* 递归思想与递归边界
* 结构体与结构体指针
* 链表结构与指针操作
* 动态内存申请与释放

后续补强：

* 文件操作
* 多文件编译
* Makefile 基础
* gdb 调试基础
* C 程序从源代码到可执行文件的过程
