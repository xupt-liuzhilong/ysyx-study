- 给加减法器加入 `Overflow` 输出

核心公式：

```text
A - B = A + (~B + 1)
S = A + (B XOR Sub) + Sub
```

其中：

| Sub  | 功能  |
| ---- | ----- |
| 0    | A + B |
| 1    | A - B |

`Cout` 面向无符号进位，`Overflow` 面向有符号补码溢出。

### F3：比较器与借位减法器

- 完成 4 位相等比较器
- 理解逐位 XNOR 判断相等的方法
- 完成 4 位无符号大小比较器
- 理解无符号比较中“高位优先决定大小”的原则
- 完成半减器 Half Subtractor
- 完成全减器 Full Subtractor
- 使用 4 个全减器级联完成 4 位借位减法器 `subtractor4`
- 区分借位减法器和补码加减法器的不同实现思路

4 位相等比较器：

```text
E0 = ~(A0 XOR B0)
E1 = ~(A1 XOR B1)
E2 = ~(A2 XOR B2)
E3 = ~(A3 XOR B3)

Equal = E3 & E2 & E1 & E0
```

无符号大小比较器的核心：

```text
从最高位开始比较
高位一旦分出大小，低位不再影响结果
只有高位相等时，才继续比较下一位
```

借位减法器模块层次：

```text
half_subtractor
→ full_subtractor
→ subtractor4
```

半减器逻辑：

```text
Diff = A XOR B
Borrow = ~A & B
```

全减器逻辑：

```text
Diff = A XOR B XOR Bin
Bout = (~A & B) | (~(A XOR B) & Bin)
```

借位减法器和补码加减法器区别：

| 电路        | 实现思路                  | 关键标志        |
| ----------- | ------------------------- | --------------- |
| add_sub4    | A - B = A + (~B + 1)      | Cout / Overflow |
| subtractor4 | A - B - Bin，逐位传递借位 | Bout            |

- 完成 4 位带写使能寄存器 `reg4`

时序逻辑学习链路：

```text
交叉配对反相器
→ S-R 锁存器
→ D 锁存器
→ 主从 D 触发器
→ 1 位寄存器

```

时序逻辑模块层次：

```text
S_R_latch
→ D_latch
→ D_flip_flop
→ reg1
→ reg4
```

D 锁存器与 D 触发器区别：

| 电路        | 更新条件          | 特点     |
| ----------- | ----------------- | -------- |
| D_latch     | WE=1 的整个时间段 | 电平敏感 |
| D_flip_flop | CLK 上升沿        | 边沿敏感 |

寄存器核心思想：

```text
EN = 1：CLK 上升沿写入新数据
EN = 0：CLK 上升沿保持原值
```

在 Logisim 中，如果反馈电路出现红线或无反应，需要检查：

```text
Simulate → Auto-Propagate
Simulate → Reset Simulation
```

同时需要先给寄存器写入一个确定值进行初始化。

计数器核心结构：

```text
reg4.Q → adder4.A
0001   → adder4.B
adder4.S → reg4.D
```

数列求和核心逻辑：

```text
cnt_next = cnt + 1
sum_next = sum + cnt_next
```

数列求和最终结果：

```text
1 + 2 + ... + 10 = 55
55 = 0011 0111
```


```text
sec_ones.EN = RUN
sec_tens.EN = sec_ones.Carry
min_ones.EN = sec_ones.Carry & sec_tens.Carry
min_tens.EN = sec_ones.Carry & sec_tens.Carry & min_ones.Carry
```

```text
MM:SS
```

1. 小模块可以组合成大模块。
2. 组合逻辑负责“计算”。
3. 时序逻辑负责“保存状态”。
4. 寄存器、加法器、MUX、比较器可以组合成更复杂的数据通路。
5. 控制信号决定数据在电路中的流向。
6. CPU 中的寄存器、PC、ALU、状态机都可以从这些基础电路逐步理解。
