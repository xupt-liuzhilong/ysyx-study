# 一生一芯学习记录

## 仓库说明

本仓库用于记录我学习“一生一芯”过程中的学习笔记、实验电路、代码文件、问题记录、运行截图和阶段复盘。

当前仓库按照学习阶段进行整理：

```text
stages/F/   F 阶段：Logisim、数字逻辑、状态机、miniRV
stages/E/   E 阶段：C 语言、HDL、Linux、二进制、PA 预学习等
resources/  官方链接、资料链接和工具文件
```
