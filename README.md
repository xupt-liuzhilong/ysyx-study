### F3：比较器与借位减法器

- 完成 4 位相等比较器
- 理解逐位 XNOR 判断相等的方法
- 完成 4 位无符号大小比较器
- 理解无符号比较中“高位优先决定大小”的原则
- 完成半减器 Half Subtractor
- 完成全减器 Full Subtractor
- 使用 4 个全减器级联完成 4 位借位减法器 `subtractor4`
- 区分借位减法器和补码加减法器的不同实现思路

4 位相等比较器：

```text
E0 = ~(A0 XOR B0)
E1 = ~(A1 XOR B1)
E2 = ~(A2 XOR B2)
E3 = ~(A3 XOR B3)

Equal = E3 & E2 & E1 & E0
```

无符号大小比较器的核心：

```text
从最高位开始比较
高位一旦分出大小，低位不再影响结果
只有高位相等时，才继续比较下一位
```

借位减法器模块层次：

```text
half_subtractor
→ full_subtractor
→ subtractor4
```

半减器逻辑：

```text
Diff = A XOR B
Borrow = ~A & B
```

全减器逻辑：

```text
Diff = A XOR B XOR Bin
Bout = (~A & B) | (~(A XOR B) & Bin)
```

借位减法器和补码加减法器区别：

| 电路        | 实现思路                  | 关键标志        |
| ----------- | ------------------------- | --------------- |
| add_sub4    | A - B = A + (~B + 1)      | Cout / Overflow |
| subtractor4 | A - B - Bin，逐位传递借位 | Bout            |

### F3：时序逻辑入门与寄存器

- 理解组合逻辑和时序逻辑的区别
- 理解交叉配对反相器可以形成稳定状态
- 完成 NOR 版本 S-R 锁存器
- 理解 S-R 锁存器的 Set、Reset、Hold 和非法状态
- 完成 D 锁存器
- 理解 D 锁存器通过 `S = WE & D`、`R = WE & ~D` 避免非法状态
- 理解 D 锁存器是电平敏感电路
- 使用两个 D 锁存器搭建主从 D 触发器
- 理解 D 触发器是边沿敏感电路
- 完成 1 位带写使能寄存器 `reg1`
- 完成 4 位带写使能寄存器 `reg4`
- 理解寄存器的作用是保存数据，而不是计算数据

时序逻辑学习链路：

```text
交叉配对反相器
→ S-R 锁存器
→ D 锁存器
→ 主从 D 触发器
→ 1 位寄存器
→ 4 位寄存器
```

时序逻辑模块层次：

```text
S_R_latch
→ D_latch
→ D_flip_flop
→ reg1
→ reg4
```

D 锁存器与 D 触发器区别：

| 电路        | 更新条件          | 特点     |
| ----------- | ----------------- | -------- |
| D_latch     | WE=1 的整个时间段 | 电平敏感 |
| D_flip_flop | CLK 上升沿        | 边沿敏感 |

寄存器核心思想：

```text
EN = 1：CLK 上升沿写入新数据
EN = 0：CLK 上升沿保持原值
```

在 Logisim 中，如果反馈电路出现红线或无反应，需要检查：

```text
Simulate → Auto-Propagate
Simulate → Reset Simulation
```

同时需要先给寄存器写入一个确定值进行初始化。
