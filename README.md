  ## 当前完成内容

  ### F1：科学提问

  - 理解科学提问的重要性
  - 学会描述目标、现象、尝试过程和自己的猜测
  - 明确以后遇到问题时要记录截图、代码、报错信息

  ### F2：Logisim 入门

  - 完成 Logisim 基本使用
  - 完成 A AND B 点亮 LED 实验
  - 完成 2 选 1 MUX 实验

  ### F3：晶体管与门电路基础

  - 理解晶体管、nMOS、pMOS、CMOS 的基本概念
  - 理解 NOT、NAND、NOR、AND、OR、XOR 的基本实现
  - 完成基本门电路的 Logisim 实现与测试

  ### F3：组合逻辑与加法器

  - 完成半加器 Half Adder
  - 完成全加器 Full Adder
  - 封装 `full_adder` 子电路
  - 使用 4 个 `full_adder` 搭建 4 位串行进位加法器
  - 封装 `adder4` 子电路

  ### F3：补码加减法器

  - 理解 4 位无符号数和 4 位补码的表示范围
  - 理解原码、反码、补码
  - 理解减法可以通过补码转换为加法：

  ```text
  A - B = A + (~B + 1)
  ```

  - 在 `adder4` 基础上实现 4 位加减法器 `add_sub4`
  - 区分 `add_sub4` 和传统全减器 Full Subtractor
  - 理解 `Cout` 和 `Overflow` 的区别
  - 给加减法器加入 `Overflow` 输出

  ### F3：简易 4 位 ALU

  - 在 `add_sub4` 基础上设计简易 4 位 ALU
  - 支持以下运算：

  | Op1  | Op0  | 功能   |
  | ---- | ---- | ------ |
  | 0    | 0    | A + B  |
  | 0    | 1    | A - B  |
  | 1    | 0    | A & B  |
  | 1    | 1    | A \| B |

  - 理解 ALU 的基本结构：

  ```text
  多条运算通路并行计算
  → MUX 根据控制信号选择一路输出
  ```

  - 使用 4 选 1 MUX 选择最终运算结果
  - 使用 Splitter 合成 `Op1 Op0` 控制信号
  - 增加 `Cout`、`Overflow`、`Zero`、`Negative` 标志位
  - 完成简易 `alu4` 电路测试
