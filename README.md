# 一生一芯学习记录

本仓库用于记录我学习“一生一芯”过程中的学习笔记、实验代码、环境配置、问题总结和阶段性复盘。

## 学习目标

- 补齐 Linux、Git、C 语言基础
- 学习数字逻辑与计算机组成原理
- 掌握 Verilog / 硬件描述语言基础
- 理解程序从 C 语言到二进制执行的过程
- 逐步完成一生一芯预学习与后续实验任务

## 目录结构

```text
logs/         每日或每周学习记录
notes/        知识点笔记
problems/     报错、卡点和解决过程
code/         自己写的小练习代码
screenshots/  环境配置和问题截图
```
